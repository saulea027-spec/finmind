# FinMind Freelancer

AI Financial Advisor specialized for freelancers in Romania & Moldova.

## Features
- 💬 AI Chat Advisor (taxes, invoicing, budgeting)
- 🧮 Tax Calculator (PFA/SRL)
- 📊 Budget Planner (50/30/20 rule)

## Deploy
1. Push to GitHub
2. Connect to Netlify
3. Add `ANTHROPIC_API_KEY` environment variable
4. Deploy!
